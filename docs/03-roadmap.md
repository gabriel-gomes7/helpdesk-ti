# Roadmap

Este documento apresenta a evolução planejada do projeto HelpDesk TI.

---

# Versão 1.0 - MVP

Objetivo: disponibilizar uma plataforma funcional para abertura e gerenciamento de chamados.

## Funcionalidades

- [ ] Criar estrutura do projeto
- [ ] Configurar Front-end (React)
- [ ] Configurar Back-end (Node.js + Express)
- [ ] Configurar PostgreSQL
- [ ] Criar API REST
- [ ] Criar formulário de abertura de chamados
- [ ] Criar painel do técnico
- [ ] Listar chamados
- [ ] Visualizar detalhes do chamado
- [ ] Alterar status
- [ ] Alterar prioridade
- [ ] Adicionar observação técnica
- [ ] Encerrar chamado
- [ ] Dashboard inicial
- [ ] Publicar aplicação

---

# Versão 1.1

Objetivo: melhorar a produtividade do técnico.

## Funcionalidades

- [ ] Pesquisa por número
- [ ] Pesquisa por colaborador
- [ ] Pesquisa por setor
- [ ] Filtro por status
- [ ] Filtro por prioridade
- [ ] Ordenação de chamados

---

# Versão 1.2

Objetivo: melhorar a experiência do colaborador.

## Funcionalidades

- [ ] Consulta de andamento do chamado
- [ ] Histórico de chamados do colaborador
- [ ] Notificações por e-mail
- [ ] Reabertura de chamados
